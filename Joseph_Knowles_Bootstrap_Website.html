Using Visual Studio, I have created several small games, as well as a WPF app that parses an XML list of US Senators, available from seante.gov, and writes out an HTML file that displays the data in a table, color-coded, and arranged by party.

C++

Using Visual Studio, I followed a game programming video series from which I learned vectors, matrices, and some of the theory behind the graphics math, as well as how to design, and implement, a reusable shell of a basic game loop framework based on a simple putPixel(int x, int y) function. By using putPixel to build ever more complex functions, like drawLine, then drawRect, then drawCube, and so on, I built a randomly generating world for a side-scrolling platformer game. Using a third-party library to read bitmaps, sequential images are looped to create an animated player character, directed with <-, ->, and spacebar to jump. Player motion is governed by algorithms that determine speed, velocity, and trajectory, and apply gravity, friction, and time.
